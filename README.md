# Проект: Работа с матрицами (учебный)
## Описание проекта
Данный проект реализует базовые операции над матрицами на языке Python.

**В нём представлены два класса:**
Matrix – работа с прямоугольными матрицами (произвольное количество строк и столбцов).

SquareMatrix – наследник Matrix для работы с квадратными матрицами (добавлена возможность создания единичной матрицы).

Проект выполнен в учебных целях для демонстрации принципов ООП, работы с одномерным представлением матрицы, а также реализации таких операций, как транспонирование, вычисление норм и вставка элементов.

## Класс Matrix
### Конструктор
```python
Matrix(rows, column, start_volume=0, end_volume=1000)
```
Создаёт матрицу размером rows × column, заполненную случайными целыми числами в диапазоне [start_volume, end_volume].

### Атрибуты
lda – количество строк (row dimension)

n – количество столбцов

matrix – одномерный список, хранящий элементы построчно

### Методы класса `Matrix`

| Метод | Описание |
|-------|----------|
| `__str__()` | Вывод матрицы в читаемом виде |
| `insert_matrix(row, column, value)` | Устанавливает значение в указанной ячейке |
| `norm(ntype="e")` | Вычисляет норму матрицы: `"e"` – евклидова, `"1"` – макс. сумма по столбцам, `"inf"` – макс. сумма по строкам |
| `transponir()` | Транспонирование матрицы |
| `_find_value(row, column)` | Внутренний метод доступа к элементу |
| `_max_sum_rows()` | Вспомогательный метод для нормы `inf` |
| `_max_sum_column()` | Вспомогательный метод для нормы `1` |
| `_evklid()` | Вспомогательный метод для евклидовой нормы |

## Класс SquareMatrix
Наследуется от Matrix.

### Конструктор
```python
SquareMatrix(rows)
```
Создаёт квадратную матрицу размером rows × rows, заполненную нулями.


### Методы
| Метод |	Описание |
|-------|----------|
| one_matrix_line()	| Преобразует текущую матрицу в единичную (на главной диагонали – 1, остальные – 0)|
Пример использования (из main.py)
```python
from matrix import Matrix
from square_matrix import SquareMatrix

matrix = Matrix(3, 4)
print(matrix)

matrix.transponir()
print(matrix)

square = SquareMatrix(3)
square.one_matrix_line()
print(square)
```
## Замечания
 - Хранение элементов осуществляется построчно в одномерном списке, адресация – row * rows + column.
 - В классе Matrix конструктор случайным образом заполняет матрицу, что удобно для тестирования.
 - Транспонирование корректно меняет размеры и перестраивает одномерное представление.
 - Класс SquareMatrix демонстрирует наследование, но в текущей версии не переопределяет методы родителя.
 - Единичная матрица создаётся явным вызовом метода one_matrix_line().
 - Строгая типизация не используется – решение учебное, упрощённое.

# Лабораторная работа: Методы сокрытия и анализа информации

## Описание
Проект реализует три основных метода работы с информацией:
- Генерация и анализ стойкости паролей
- Восстановление паролей по хешу
- Сокрытие данных в изображениях методом LSB

## Структура проекта
- `module_1.py` - генерация паролей, вычисление энтропии, XOR шифрование
- `module_2.py` - перебор 4-значных паролей по хешу
- `module_5.py` - LSB стеганография (сокрытие и извлечение текста в изображениях)
- `main.py` - пример использования module_1

## Модуль 1 (module_1.py)
**Функции:**
- `generate_password(length)` - генерирует случайный пароль заданной длины из букв, цифр и символов
- `calculate_entropy(password)` - вычисляет энтропию пароля на основе используемых символов
- `xor_encrypt(text, key)` - шифрование текста методом XOR
- `xor_decrypt(encrypted_bytes, key)` - дешифрование XOR

## Модуль 2 (module_2.py)
**Функции:**
- `hash_encrypt(hash_value)` - перебирает все 4-значные пароли (0000-9999) и возвращает пароли, чей хеш (сумма ASCII кодов % 10000) совпадает с заданным значением

## Модуль 5 (module_5.py)
**Класс LSBSteganography:**
- `text_to_bits(text)` - преобразование текста в битовую строку с добавлением длины сообщения
- `bits_to_text(bits)` - обратное преобразование битов в текст
- `hide_message(image_path, message, output_path)` - сокрытие сообщения в изображении путем замены младших битов пикселей
- `extract_message(image_path)` - извлечение скрытого сообщения из изображения

## main.py
Демонстрирует генерацию пароля и вычисление его энтропии.
